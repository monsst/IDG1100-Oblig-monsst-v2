24. Create a step by step tutorial on how to create a cronjob that prints “Task scheduled at” + DATE. Cronjob is scheduled every 30 minutes on today’s date (only today).

First you use the command crontab -e to make a cronjob. After that you add inn the time you want it to execute. In this example that would be */30 * 27 11 * . (This was written the 27th of November.) On the right side of that you add your command: echo "Task scheduled at $(date)." Continuing to the right you add where you want the text to be echoed to: >> /home/username/something/somthing.etc Then you save your cronjob, and exit. Now you should check if the cronjob is running: service cron status. Make sure that you give your cronjob the permissions it need with the chmod command. Now to make sure it is running you can write: service cron start. It should now work.
